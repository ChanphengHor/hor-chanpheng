Setup CV Data in Firestore

Setup complete! Your CV data is now in Firestore.

You can now view your CV, which will load data from Firestore.

Data Structure

The following data structure will be created in Firestore:

Collection: 'cv'
Document: 'main'
Fields:
- profileImage: URL to profile image
- name: Your name
- title: Your professional title
- profile: About text
- phone, email, website, address: Contact info
- education: Array of education entries
- expertise: Array of skills/expertise
- languages: Array of languages
- work: Array of work experience entries
- references: Array of references
